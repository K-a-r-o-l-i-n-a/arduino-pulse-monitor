# Arduino Pulse Monitor

Arduino alapú szimulált pulzusmonitor projekt LCD kijelzővel és LED-es visszajelzéssel.

## Funkciók
- BPM kijelzés LCD kijelzőn
- LED-es pulzustartomány jelzés
- Potméteres pulzus szimuláció
- Tinkercad alapú megvalósítás

## Felhasznált eszközök
- Arduino Uno
- LCD 16x2 I2C
- Potméter
- LED-ek
- Ellenállások
- Breadboard
